# Fluffy

Fluffy is a framework designed for the rapid development of services in
a service-oriented architecture using RabbitMQ.

## Installation

Add this line to your application's Gemfile:

    gem 'fluffy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluffy
